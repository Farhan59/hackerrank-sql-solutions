# HackerRank SQL Solutions

A collection of SQL solutions maintained for active learning and technical revision. This repository documents my daily progress during an intensive skill-building phase in March 2026.

## 🛠️ Development Workflow
To ensure code quality and logic accuracy, every solution follows a manual verification process:

* **Manual Drafting**: Queries are initially written in a text editor to reinforce syntax mastery without IDE assistance.
* **Strict Validation**: Each script is tested against **MySQL** standards and verified through all HackerRank test cases before being committed.
* **Clean Versioning**: Files follow a standardized naming convention (`query_XX_name.sql`) for better maintainability.
